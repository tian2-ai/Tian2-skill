**Source:** `/Volumes/Mac-Mini/workspaces/tian2-edu/Competitions/isef-research-playbook/05-analysis/project-workflow-and-category-map-2026-05-01.md` (verified extract from official 2026 Rules Book PDF).

Use this map to (a) infer a topic's likely ISEF category in `score` mode, (b) suggest 2–3 best-fit categories in `discover` mode based on a student profile.

## §1 — Selection heuristics (interest → category)

When the student's profile lands cleanly in one place, recommend it. When it bridges multiple, recommend 2–3 with one-sentence justifications. Never recommend more than 3 without narrowing first.

| Biology + microscope + school lab | CELL, BMED, MCRO, ANIM |
| Biology + computation + datasets | CBIO, BMED, BCHM |

| Chemistry + lab + analytical | CHEM, BCHM, MATS |

## §3 — Cross-category bridge guidance (for T1.1.b)

The most common cross-category bridges that score well on T1.1.b "cross-disciplinary bridge" and unlock the M4 cross-category importation bonus:

| Bridge | Example shape | Why it works |
|--------|---------------|--------------|
| PHYS ↔ BMED/CELL | Apply statistical-mechanics or condensed-matter framing to biology (e.g., PHYS021's origami → tissue folding) | Physics formalism + biological substrate creates a rare combination |
| MATH ↔ CBIO/SFTD | Graph theory or topology applied to biological networks or code | Pure-math methods rarely meet messy data |
| CHEM ↔ EAEV/ENEV | Analytical chemistry tools for environmental forensics | Lab technique × field problem |
| ROBO ↔ ANIM/BMED | Bio-inspired robotics; assistive devices | Visible prototype + biological motivation |
| MATS ↔ EGSD | Novel material applied to energy storage / generation | Sustainability narrative + measurable performance |
| BEHA ↔ CBIO/SFTD | Computational social science / NLP on behavior | Quant + behavior is rare in HS |

**Note for `score` mode:** if a topic clearly spans 2+ categories, the student must pick one for ISEF — that's a rule (Book §1, line 137). But the cross-category framing still earns T1.1.b points; surface this and let the student choose strategically.
