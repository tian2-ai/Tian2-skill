## Refresh protocol

1. Run `WebFetch` on each URL with prompt: "List all current dates, deadlines, and qualification thresholds. Quote verbatim."
2. Update timelines in `SKILL.md` §A-D to match.
3. Update the `last verified` column with today's date.
4. If a source returns a 404 or is reorganized, replace with the new canonical URL — never silently lose a citation.

## Why these specific sources

- **regeneronisef.org.cn** is the official Regeneron-sponsored ISEF China entry portal — operated by ISEF's Chinese partner. Authoritative for the CN-track pathways.
- **societyforscience.org** is the parent body running ISEF globally.
- **Hanlin** and **Jingsailian** are CN-track agencies that publish timeline summaries; treat as secondary verification, not primary.

Never use individual blog posts, student-forum summaries, or social-media screenshots as source — too easy to get year-of-cycle confusion.

## Known unknowns

The exact mapping of the 8 CASTIC regions to specific provinces is published on
https://www.cyscc.org/castic/ but not surfaced in the data fetched 2026-05-27. Future refresh
should capture this mapping.

The advancement rates from CASTIC → ISEF China team selection are not publicly disclosed. Hanlin
quotes ~5-10% historical rate but this is unofficial.
