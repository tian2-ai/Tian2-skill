# ISEF Poster Design Best Practices

> Pre-researched reference for poster evaluation. Use this checklist when scoring the Presentation criterion.

---

## Display Requirements (Hard Rules)

| Dimension | Maximum |
|-----------|---------|
| Width | 122 cm (48 in) |
| Depth | 76 cm (30 in) |
| Height (floor to top) | 240 cm (94.5 in) |

### Must Display
- Official Abstract with SRC stamp (displayed vertically)
- Project Set-up Approval Form (signed)
- Form 1C (if institutional research)
- Form 7 (if continuation project)

### Prohibited
- QR codes, personal addresses, email, phone numbers
- Commercial logos, institutional crests, trademarks
- Published papers (keep in notebook only)
- Awards/medals (except ISEF medals worn)
- USB drives, brochures, business cards, food
- Biological materials, chemicals, glass, flames, sharp items

### Graphics Citation
Every graphic must be individually cited (APA with URLs).

## Layout (Tri-Fold)

```
+-------------------+----------------------------+-------------------+
|   LEFT PANEL      |      CENTER PANEL          |   RIGHT PANEL     |
|  Introduction     |  TITLE (full width)        |  Results (cont.)  |
|  Background       |  Abstract                  |  Discussion       |
|  Hypothesis /     |  Methods                   |  Conclusion       |
|  Research Q       |  Results (primary visuals)  |  Future Work      |
|                   |                            |  References       |
|                   |                            |  Acknowledgments  |
+-------------------+----------------------------+-------------------+
```

### Space Allocation

| Section | Share | Priority |
|---------|-------|----------|
| Results / Data Visualizations | ~40-50% | Highest |
| Title + Abstract | ~10-12% | High |
| Introduction / Background | ~10-12% | Medium |
| Methods | ~8-10% | Medium |
| Discussion / Conclusion | ~10-12% | High |
| References + Ack + Future Work | ~5-8% | Required |
| White space | ~15-20% | Non-negotiable |

## Typography

| Element | Minimum | Recommended | Readable from |
|---------|---------|-------------|---------------|
| Title | 72 pt | 84-150 pt | 10-15 ft |
| Section headers | 36 pt | 48-72 pt | 6-8 ft |
| Body text | 24 pt | 24-32 pt | 3-4 ft |
| Figure captions | 18 pt | 20-24 pt | 2-3 ft |
| Photo credits | 10 pt | 12-14 pt | Close-up |

- Sans-serif: Arial, Helvetica, Calibri, Tahoma
- Max 2 font families
- Left-align body; center titles/headers only

## Color

- 2-3 colors max (plus black/white)
- Dark text on light background (high contrast mandatory)
- Recommended: navy + white + gold; charcoal + gray + teal
- Avoid: neon, red-on-blue, patterned backgrounds behind text

## Word Count Targets

| Section | Target |
|---------|--------|
| Title | 10-15 words |
| Abstract | 150-250 words |
| Introduction / Background | 80-120 words |
| Hypothesis / Research Q | 30-50 words |
| Methods | 80-150 words |
| Results | 50-100 words (figures carry it) |
| Discussion | 100-150 words |
| Conclusion | 60-100 words |
| Future Work | 30-60 words |
| Acknowledgments | 30-50 words |
| **Total (excl. abstract)** | **500-800 words** |

## Essential Elements Checklist

### Required by ISEF
- [ ] Title (clearly identifies project)
- [ ] Student name, school
- [ ] SRC-stamped abstract
- [ ] Approval Form visible
- [ ] Individual graphic citations (APA)

### Strongly Recommended (Judged)
- [ ] Research question / hypothesis stated explicitly
- [ ] Background / Introduction
- [ ] Materials and Methods
- [ ] Results (visual-dominant)
- [ ] Data tables and/or graphs
- [ ] Discussion / Analysis
- [ ] Conclusion (directly answers hypothesis)
- [ ] Limitations / Sources of Error
- [ ] Future Work
- [ ] References / Bibliography
- [ ] Acknowledgments

### Data Quality Markers (Physics / Quantitative)
- [ ] Error bars on every plot (labeled: SD, SEM, or 95% CI)
- [ ] Axis labels with units on every graph
- [ ] Figure number + caption below every figure
- [ ] n= reported for every dataset
- [ ] Statistical test results (p-value, CI, R²) where applicable
- [ ] Uncertainty to 1 significant figure
- [ ] For n < 10: individual data points plotted, not just mean

## Common Mistakes

### Text
| Mistake | Fix |
|---------|-----|
| Walls of text | Bullet points; 500-800 words total |
| Font below 24pt | 24pt minimum body |
| Too many fonts | Max 2 families |
| Repeating figure info in text | Let figure speak; caption adds context |

### Data / Figures
| Mistake | Fix |
|---------|-----|
| Missing axis labels/units | Every axis: label + unit |
| No error bars | Add; state what they represent |
| Low-resolution images | 300+ DPI export |
| No figure captions | Numbered caption below every figure |
| Uncited figures | APA citation with URL |

### Layout
| Mistake | Fix |
|---------|-----|
| No reading order | Left-to-right, top-to-bottom; numbered headers |
| No white space | 15-20% intentional breathing room |
| Missing visual hierarchy | Title > headers > body, no exceptions |

### Content
| Mistake | Fix |
|---------|-----|
| Conclusion doesn't answer hypothesis | State: "supported/not supported because..." |
| No limitations | Add — signals maturity |
| No references | Must cite literature |
| No future work | Even one sentence shows depth |

## What Makes a Poster Stand Out

- **One hero figure** — dominant visualization capturing core finding
- **Consistent visual language** — same color = same variable across all graphs
- **Professional figures** — matplotlib/R/MATLAB, not Excel defaults; vector or 300+ DPI
- **Limitations section present and specific** — not generic "human error"
- **Title as finding/question** — not just topic
- **Interview-ready design** — poster raises questions judges want to ask, doesn't exhaustively answer everything

**Remember:** Interview (25 pts) is worth 2.5x the poster (10 pts). Design to make the interview conversation deep.

## Print Checklist
- [ ] All images 300+ DPI
- [ ] No stretched/pixelated images at print size
- [ ] Fonts embedded
- [ ] Contrast: WCAG AA minimum (4.5:1 for body text)
- [ ] Proofread by someone unfamiliar with project
- [ ] Spell-check complete
- [ ] Citations verified
- [ ] Dimensions within 122cm × 240cm
- [ ] Draft printed at 50% scale to check layout
